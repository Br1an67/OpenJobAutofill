# 简历资料小抄

一个 Brave/Chrome Manifest V3 扩展，用右侧栏展示本机 Markdown 简历小抄，并提供隐私优先的页面扫描、填充草稿和确认后写入。

## 功能

- 右侧栏展示资料小抄，支持收起成窄条、展开、隐藏。
- 一键流程会扫描页面、生成填充草稿，把你勾选的项目写入当前网页表单。
- 可选 API 只接收页面字段和本地资料字段目录，不接收 Markdown 里的具体简历值。
- 右侧栏状态和草稿会保存在本机扩展 session 中，popup 关闭后不会丢。
- 内置 zhiye、HotJob、Ant Design、Element UI 等常见招聘表单/组件的适配启发式。
- 设置页用 Markdown 保存资料，不需要编辑 JSON。
- Markdown 按 `## 大类` 分组，侧边栏先显示分类目录，点进分类后查看具体内容。
- 支持搜索字段和值，例如 `手机`、`学校`、`证书`。
- 详情内容可直接用鼠标选中复制；也可以复制整个分类。
- 旧 JSON 资料可以导入，会自动转换为 Markdown。
- 资料只保存在 `chrome.storage.local`，不会同步到云端。
- 不自动提交，写入前会让你确认。

## Markdown 格式

资料结构建议参考塔塔网申式在线简历：`基本信息`、`求职意向`、`教育经历`、`实习经历`、`工作经历`、`项目经历/实践活动`、`学生工作`、`奖惩情况`、`外语能力`、`计算机技能`、`证书`、`家庭情况`、`培训经历`、`论文和著作`、`专利`、`自我描述`、`有关声明`。这样同一份 Markdown 可以覆盖大多数校招/社招网申字段。

```markdown
# 简历资料小抄

## 基本信息
- 姓名：李明
- 手机号码：13900000000
- 电子邮箱：liming@example.com

## 教育经历
### 教育经历 1
- 毕业院校：星河大学
- 专业：计算机科学与技术
- 学历：本科

## 求职意向
- 意向岗位：软件开发工程师
- 期望工作城市：上海

## 实习经历
### 实习经历 1
- 公司：星桥科技有限公司
- 部门：平台研发部
- 职位：软件开发实习生
- 工作内容：参与招聘系统和数据看板开发。
```

后续要追加内容，直接在对应大类下面新增一行：

```markdown
- GitHub：https://github.com/example
```

## 安装到 Brave

1. 打开 `brave://extensions/`。
2. 开启右上角 `Developer mode`。
3. 点击 `Load unpacked`。
4. 选择本目录：`job-form-ai-autofill`。
5. 点击扩展图标，先进入 `设置` 保存 Markdown 资料。

## 使用流程

1. 在设置页维护并保存 Markdown 小抄。
2. 打开招聘网站表单页面。
3. 点击扩展图标，选择 `开始填写`，或先选择 `打开右侧栏`。
4. 侧边栏会生成草稿，检查勾选项后确认写入。
5. 如果识别不准，就回到小抄分类，点进 `基本信息`、`项目经历`、`实习经历`、`社团工作`、`学生工作`、`奖惩情况` 等分类手动参考。
6. 详情内容可直接选中文字复制，也可以点击 `复制本类`。

```text
job-form-ai-autofill/

  src/
    background.js   # 本地存储、可选 API 兼容逻辑
    content.js      # 右侧栏、Markdown 小抄渲染和复制
    popup.html      # 右侧栏入口
    popup.js
    popup.css
    options.html    # 设置页
    options.js
    options.css
```
